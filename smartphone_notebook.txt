

### smartphone convert to notebook ###

1.Install Termux (google player)

2.Setting Termux-setup-storage
  https://wiki.termux.com/wiki/Termux-setup-storage
  - 안드로이드 앱 설정 - termux 앱 권한 저장공간 허용 
  Android 11
	You may get "Permission denied" error when trying to access shared storage, even though the permission has been granted.
	Workaround:
	Go to Android Settings --> Applications --> Termux --> Permissions
	Revoke Storage permission
	Grant Storage permission again
	
  $ termux-storage-storage
	
3.mkdir ubuntu
  - Termux execute
  - mkdir ubuntu

4.pkg command study & install vim
  https://admion.net/port-and-pkg/
  - pkg install vim

5.Install Andronix (google player) 

6.Install bVNC: Secure VNC Viewer (google player)

7.Install ubuntu
  - Termux execute
  - cd ubuntu
  - pkg update -y && pkg install curl proot tar -y && curl https://raw.githubusercontent.com/AndronixApp/AndronixOrigin/master/Installer/Ubuntu20/ubuntu20-xfce.sh | bash
    (Andronix execute -> Ubuntu -> Install -> Ubuntu 20.04 -> XFCE : command copy)
    
    위의 명령어 실행시 저장소 관련 오류 발생 시 아래 명령어 실행 후 재 실행
    pkg remove game-repo
    pkg remove science-repo
    pkg update 
    
8.Install Openbox
  - Termux execute
  - cd ubuntu
  - pkg update -y && pkg install curl proot tar -y && curl https://raw.githubusercontent.com/AndronixApp/AndronixOrigin/master/Installer/Ubuntu20/ubuntu20-openbox.sh | bash
    (Andronix execute -> Ubuntu -> Install -> Ubuntu 20.04 -> Window Manager -> Openbox : command copy)
  - ./start-ubuntu20.sh
  - openbox.sh (chmod 755)
  - ./openbox.sh
   
	cf) 설치 중 오류 발생 시 아래의 dns 정보 수정
	    - dns setting
	    - vi /etc/resolv.conf
	    nameserver 1.1.1.1
	    nameserver 8.8.8.8
	    nameserver 8.8.4.4
  - 설치 종료 시점 vncserver 패스워드 설정

9.vnc execute shll 
   - vncserver 해상도 설정
   - xrandr -s 1920x1080
   - vi vnc.sh
    vncserver-stop && vncserver-start
   - chmod 755 vnc.sh
   - ./vnc.sh
   
   이미 vnc는 설치되어 있음 설치 안되어 있으면 아래 설치 구문으로 설치
   apt-get install tigervnc-standalone-server
   
   위와 같이 개발 수동 실행 하거나 
   vi ~/.profile 파일 하단에 아래 구문 추가 하여 자동 실행 처리 
   vncserver-stop & vncserver-start
